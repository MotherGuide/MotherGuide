# MEASUREMENT THEORY
---


## Overview

This document gives a description of the modules of the MotherGuide platform which include Authentication, Tips and Doctor Modules

---

## Authentication (Login and Sign up) Modules

| Component | Metric Performed | Scale Type | Values |
|---|---|---|---|
| Pregnancy Duration | Weeks of pregnancy / Trimester | Ratio Scale | 0 to 40 |
| Logging and Signing up | Loading Time | Absolute Scale | milliseconds from 0 (e.g. 185ms) |

---

## Tips (Nutrition, Warning Signs, Preparing for Delivery, Baby Care) Module

| Component | Metric Performed | Scale Type | Values |
|---|---|---|---|
| User interaction | Likes, Dislikes, Comments and Views | Absolute Scale | Numbers Starting from 0 |

---

## Doctor Module

| Component | Metric Performed | Scale Type | Values |
|---|---|---|---|
| Question Input | Loading time | Absolute Scale | milliseconds from 0 (e.g. 193ms) |
